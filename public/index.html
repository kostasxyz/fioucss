#Grid

Grid widths are assignd by intervals of 5 ( 5%, 20% 35%) and 1-3 ( 1 third = 33.3333%) and 2-3 (2 thirds = 66.6666%).
There is only one media query max-width: 1199px and min-width: 1200px. Basicaly all screens below 1200 are considered small screens.
All columns are 100% size below 1200px. You can add the .f_row-sm and .f_flex-sm with @sm in the f_col-* class (f_col-50@sm) to have mobile columns.

#Level

The level component is just a simple flex container that makes easy to create a nav menu, a toolbar etc.
It has a left item, a right item to easily align logos, action buttons etc.
